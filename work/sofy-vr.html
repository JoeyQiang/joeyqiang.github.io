sofy vr attracting more users to strengthen the brand identity
SOFY VR is a Virtual Reality shooter game with many different types of food to consume, which are considered bad for girls when experiencing menstruation. The main goal is to shoot as fast as possible all different monsters on the VR device. Through playing the game, users can have more recognition for the brand of SOFY.

Requirements

Rabbit
  • Add a new entry of the game to the current app
  • Responsible for the mobile interface design of none VR part
  • Highlight the Game Start after user get into the game, playing it as soon as possible
  • Provide text-based tutorial for user to better understand how to play the game
  • Build a E-Commerce which user can redeem coupon using points they get from playing the game
  • Create a friendly experience for user when transferring from none VR part to VR part

Iteration

Home Page

Change the layout of home page to highlight START button, making user play the game without thiking

Selecting Levels

Change the way to select levels from manual control to eyes control, making user easy to swtich levels without taking VR glasses down

Tutorial Page

Change the browsing gesture of Tutorial page from clicking left & right to scrolling up & down, expediting user’s browsing speed

Game Over

Enlarge the size of buttons to make them easier to focus. Add spinning & fading animation to cartoon characters to make interface more vivid

UI

Conclusion

  • Quicker design turnaround. Push clients to give feedback as soon as possible and ask them to sign off every updates
  • Keep everyone on the same page. Discuss with PM & UI designers about every changes everytime. Notes all changing details in email in order to review later
  • Always make clients happy. Follow clients’ design guide rigorously and try best to meet their requirements though sometimes they come up with unreasonable ideas
