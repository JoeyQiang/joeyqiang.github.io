Summary

Overall Findings Recommendations
The structure of information is not well organised and repetitive Refine the IA via the personas and user journey
The key functions (register, free sample, coupon, store finder) require too many steps Simplify the procedure of key functions. Review on missed opportunities to collect information and deliver more targeted/relevant messaging
Screens lack connection with each other and visual design across the screens are inconsistent Update visual design with consistency across the screens to be more modern and familiar to users
Pages load slowly and are inconvinient to browse due to heavy images Reduce the usage of heavy background images and optimize browse experience

Personas

Amber

First Time User
  • Have never used ACUVUE contact lenses before or any other brand lenses
  • Know little about the contacts
  • Need to wear contacts for special occasions

John

Experienced User
  • Have been using ACUVUE contacts before
  • Have never used ACUVUE WeChat account
  • Need to purchase contacts regularly. Plan to wear contacts everyday

Linda

Loyal User
  • ACUVUE WeChat account user
  • Need to purchase ACUVUE contacts regularly
  • Plan to wear ACUVUE contacts everyday

User Journey

Amber

First Experience

  • None

Offline

  • The clerk gives Amber an optometry and choose the contacts that fit her requirements
  • The clerk recommend the official account to Amber

Online

  • Amber follow the official account then register to get points and coupons
  • She can use the points to get gifts
  • Also Amber can browse more products and discover nearer stores

John

First Experience

  • John finds out he will get more coupons and benefits if he registers as a member of ACUVUE.
  • He follows the account then registers

Offline

  • John comes to the store to buy the contacts with a coupon.
  • He finds out from the clerk that the account can also accumulate points

Online

  • Use the points to redeem gifts
  • He can browse products and get free samples
  • Also he can find nearer stores

Linda

First Experience

  • Linda has ran out of her contacts. She needs to buy some new ones.
  • Find easily stores from the official account.

Offline

  • She buys the contacts then ask the clerk how many points she can get
  • Linda asks the clerk are there any new free samples she can try

Online

  • Use the points to redeem gifts
  • Pays attention to the new discounts
  • Pays attention to the new products

Updated UX

AI

Visual AI

A1 Products List

Improvements:

  • Keep all products within one page, which is more friendly for user to browse. And put Free Sample part as a banner at the bottom of the page
  • Make free sample&coupon links to an attractive button, call to action
  • Classify coupons to limited & unlimited ones, which motivates user to use coupon asap

A2 Guide

Improvements:

  • Keep details of the answer in the same page with the question, which can be collapsed & expanded
  • Add product recommendation relating to question at the bottom, which can increase the selling conversion rate

B1 Find Stores

Improvements:

  • Use Map View to give user a clear & straightforward finding experience
  • Show surrounding stores in default based on user’s location, and also provide search function for user to find specified store. Always put a default detail info on the Find Store page (could be the nearest one)
  • After user select the store, they can view its info and use WeChat Route API to navigate to the destination

B2 Coupons

Improvements:

  • Show coupons that user already have
  • Send user coupons from time to time, and prompt user when they land on the page
  • Give user a get coupon link at the page bottom if clients want to get some useful information from user

C1 Points & Redemption

Improvements:

  • Reorganize the content, give a bottom link jumping to Getting Points Page
  • Add progress bar to each product, prompting user how far they can get the product, so encourage user to get points and exchange products
  • After exchange a product successfully, user can view redemption history on the account page

C2 My Account

Improvements:

  • Change the tab name to a more understood one
  • The first time when user land on the page, ask user to bind account, then to sign up so they can use 100 free points
  • Add Coupons Information & Redemption History to the account page after user has signed up
